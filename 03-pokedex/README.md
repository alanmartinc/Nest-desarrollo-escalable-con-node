# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

## Stack usado
* Nest
* Postman
* MongoDB
* TablePlus
* TypeORM
* Docker
* Websockets
